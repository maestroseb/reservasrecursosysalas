Activar Sistema

Sigue estos 3 pasos

1

Abrir Apps Script

En el menú superior, ve a:

Extensiones → Apps Script
2

Crear Implementación

Haz clic en el botón azul:

Implementar → Nueva implementación

⚙️ Configura así:

  • • Tipo: Aplicación web
  • • Ejecutar como: Yo
  • • Acceso: Cualquier usuario
3

Copiar y Abrir

Google te dará una URL (enlace web).

Copia esa URL, abre una pestaña nueva en tu navegador y pégala allí.

Al abrir el enlace, verás el asistente final para crear las pestañas y configurar tu cuenta de Admin automáticamente.
